## Installation

Use the package manager [pnpm](https://pnpm.io/ru/) to install all dependencies.

```bash
pnpm install
```

## Usage

To run project start command

```bash
pnpm run dev
```

[Backend](http://localhost:3005/) will start at the

```htpp
http://localhost:3005/
```

[Frontend](http://localhost:5173/) will start at the

```htpp
http://localhost:5173/
```
